## 🎯 依使用場景選擇

### 我是新用戶，想快速開始

1. 閱讀 [快速入門指南](QUICK_START.md)
2. 選擇版本：[Heavy vs Tiny](TINY_VS_HEAVY.md)
3. 依照安裝指引安裝應用程式
4. 開始使用！

### 我想了解特定功能

1. 查看 [功能完整參考](FEATURES_REFERENCE.md)
2. 找到您需要的功能章節
3. 依照步驟操作

### 我遇到問題了

1. 先查看 [常見問題 FAQ](FAQ.md)
2. 如果沒有找到答案，查看 [疑難排解指南](TROUBLESHOOTING.md)
3. 仍未解決？前往 [GitHub Issues](https://github.com/ChengTingFung-2425/robot-command-console/issues)

## 🔧 依使用環境選擇

### 桌面應用程式用戶

- **Heavy (Electron) 版本**
  - 參考：[快速入門指南](QUICK_START.md) 的 "Heavy 版本" 章節
  - 完整 UI 互動體驗

- **Tiny (PyQt) 版本**
  - 參考：[Tiny 安裝指引](TINY_INSTALL_GUIDE.md)
  - 輕量快速部署

### 終端機用戶

- **TUI（Terminal UI）模式**
  - 參考：[TUI 使用指南](TUI_USER_GUIDE.md)
  - 適合無頭伺服器或遠端 SSH

### Web 介面用戶

- **WebUI 模式**
  - 參考：[WebUI 使用指南](WEBUI_USER_GUIDE.md)
  - 瀏覽器存取，支援多人協作
