# Tiny vs Heavy 版本選擇指引

> **最後更新**：2025-12-10  
> **適用版本**：v1.0.0+

---

## 概述

Robot Command Console 提供兩個版本供使用者選擇：

- **Heavy (Electron)** - 完整功能版本
- **Tiny (PyQt)** - 輕量化版本

兩個版本功能相容，可根據您的需求與環境選擇合適的版本。

---

## 快速比較

| 特性 | Heavy (Electron) | Tiny (PyQt) |
|------|------------------|-------------|
| **安裝包大小** | ~150-300MB | ~40-60MB |
| **記憶體佔用** | ~300-500MB | ~150-250MB |
| **啟動速度** | 2-5 秒 | 1-3 秒 |
| **前端框架** | React (客戶端渲染) | Jinja2 (伺服器端渲染) |
| **WebView 引擎** | Chromium (內嵌) | QtWebEngine (系統) |
| **開發工具** | ✅ 完整 DevTools | ⚠️ 有限 |
| **UI 互動** | ✅ 豐富 | ⚠️ 基礎 |
| **離線支援** | ✅ 完整 | ✅ 完整 |

---

## 使用場景建議

### 選擇 Heavy (Electron) 如果您：

1. **是開發者或進階使用者** - 需要完整的開發工具
2. **追求最佳使用者體驗** - 喜歡流暢的動畫與互動
3. **硬體資源充足** - RAM ≥ 4GB
4. **開發/測試環境** - 需要頻繁修改與測試

### 選擇 Tiny (PyQt) 如果您：

1. **在資源受限環境** - RAM < 4GB，磁碟空間有限
2. **生產環境部署** - 需要穩定可靠的運行
3. **只需要核心功能** - 基礎的 UI 互動已足夠
4. **邊緣運算設備** - 工業電腦、嵌入式系統、IoT 閘道
